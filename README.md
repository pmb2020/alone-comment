# zxcomment
真心评论系统，简单、免费、无广告。

开发记录
思路：
点击发表评论，将访客信息存入cookie（昵称、邮箱、网址），

点击回复，判断cookie是否存在，不存在提示用户生成自己的id通行证，生成后可评论
